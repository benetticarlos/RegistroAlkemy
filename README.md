# RegistroAlkemy

Gracias por ver mi proyecto. Es un CRUD para organizar los ingresos y gastos de un usuario, está creado con React, MySQL, NodeJS y express.

Este es mi primer proyecto usando React aprendiendolo de forma autodidacta, por lo que puede tener errores o no ser optimo.

Al momento de subida esta app tiene dos inconvenientes: El primero, por mi falta de experiencia en React, no supe como interactuar dos componentes entre si por lo que 
los registros no se actualizan automaticamente, lo debe hacer el usuario con el boton "actualizar" debajo del balance total. El otro inconveniente surge con la API 
que hice, la cual no pude conectarla al cliente por un problema de CORS (desconocía esto, en mis anteriores APIs nunca me pasó). El servidor puede cargar y requerir datos de la base pero no desde el cliente, el cual terminé usando el local storage.
De todas maneras subo la App porque me sirvió de ejercitación y aprendí mucho usando React, con la certeza de que las proximas aplicaciones seran cada vez mas eficientes.

Muchas gracias.
